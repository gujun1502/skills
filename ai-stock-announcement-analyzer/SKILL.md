# AI 股票公告分析器（盘前）

用 akshare 抓取 A 股上市公司公告，分类、抽取要点、做利好/利空初判，帮助用户在**盘前**快速过一遍信息。

## 适用场景

- "今天/最近 XX(代码或名称)有什么公告？" → 个股公告
- "盘前帮我扫一下全市场重要公告" → 全市场公告
- "这条减持/重组/业绩预告公告怎么解读？" → 单条公告解读
- "帮我看看 600519 最近的公告利好还是利空" → 个股 + 倾向判断

## 工作流程

1. **确定目标**：个股(需要 6 位代码，如 `600519`/`000001`；只给名称时先用 `scripts/announce.py --resolve 名称` 转代码) 还是全市场。
2. **运行脚本**取数：
   ```bash
   # 个股最近公告（默认近 30 天）
   python scripts/announce.py --symbol 600519
   # 个股 + 指定起始日期
   python scripts/announce.py --symbol 000001 --start 20260601
   # 全市场某天的公告（按类型可选）
   python scripts/announce.py --market --date 20260617
   # 名称转代码
   python scripts/announce.py --resolve 贵州茅台
   ```
   脚本输出 JSON（公告标题、类型、日期、链接）到 stdout。
3. **分类与解读**：拿到 JSON 后，由你(Claude)按下表归类并逐条给出**倾向 + 一句话理由**。不要编造公告里没有的数字。
4. **汇总**：输出一张表(标题 / 类型 / 日期 / 倾向 / 关注理由) + 3~5 条"盘前重点关注"。
5. **免责声明**：结尾固定加一句"以上为公开信息整理与启发式判断，仅供研究参考，不构成投资建议。"

## 公告分类与倾向启发式

| 类型关键词 | 默认倾向 | 说明 |
|---|---|---|
| 业绩预增/扭亏/中标/收购/增持/回购/股权激励 | 偏利好 | 需看幅度，预增需对比预期 |
| 业绩预减/亏损/减持/质押/被处罚/立案/商誉减值/担保 | 偏利空 | 减持看比例与主体 |
| 重组/资产注入/定增/可转债 | 中性偏事件驱动 | 看条款与摊薄 |
| 股东大会/补充更正/日常关联交易/季报披露时间 | 中性 | 例行 |

> 倾向只是初筛，**必须结合公告正文金额、比例、对比预期**再判断；信息不足时标注"需看正文"。

## 注意事项

- akshare 走公网，首次/高峰期可能较慢或偶发失败；脚本已带重试与友好报错，失败时把报错原样告诉用户并建议稍后重试。
- 接口数据有延迟，盘前以"昨日收盘后~今晨"披露为主。
- 只覆盖 A 股(沪深北)。
