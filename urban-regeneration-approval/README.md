# urban-regeneration-approval

针对中国大陆既有建筑改造（解危加固/外立面/机电更新等）项目的设计与报建辅助 Claude Code skill。

## 安装位置

`~/.claude/skills/urban-regeneration-approval/`（已就位于本目录）

## 触发方式

向 Claude 提及既有建筑改造、老楼加固、报建审图、原图缺失、审图意见、跨城市政策、业主沟通函等任意一项即可。

## 知识底本来源

- 案例：某银行奉化支行大楼解危加固外立面维修及配套工程
- 政策基础：《宁波市既有建筑改造消防技术指南》（2024.1）
- 实战经验：建筑/结构/给排水/暖通/电气五专业的真实审图意见与设计回复

## 使用提示

本 skill 内置的政策细节基于宁波/江浙沪经验。跨城市使用前必须：

1. 确认项目所在城市现行版本的《既有建筑改造技术指南》
2. 与本地审图机构、消防大队进行预沟通
3. 不得直接套用一城市经验到另一城市
