# 决策框架：四块研究 → 五因子加权 → 买卖结论

本文件定义 stock-tp 的评分口径、权重语义、决策映射与风险否决。目标是让每一次
"买入 / 卖出 / 持有"的结论都**可视、可追溯、权重可调**。

---

## 1. 四块研究 ↔ 五因子映射

用户研究分四块，落到引擎是五个 0–100 的因子子分（资金面与技术面一同支撑第三块）：

| 研究块 | 因子(key) | 子分来源 | 谁来评 |
|--------|-----------|----------|--------|
| ① 信息/消息/新闻面 | `news` 消息面 | Anspire Search 检索 → 拆解"预言" | **Claude**（脚本占位 50） |
| ② 情绪面 | `sentiment` 情绪面 | `sentiment.py`（涨跌停家数 / VIX / 大盘趋势） | 脚本 |
| ③ 财报/行业/盈利点 | `fundamental` 基本面 | `fetch_fundamentals.py` 的 P0 语义分 | 脚本 + Claude 校正 |
| ③ 同上（量价/主力） | `technical` 技术面、`capital` 资金面 | 指标 / 主力净流入 | 脚本 |
| ④ 综合分析与预测 | —— | 加权综合 + Claude 叙述 | 引擎 + Claude |

## 2. 默认权重（可配置）

```
news        消息面   0.22
sentiment   情绪面   0.13
fundamental 基本面   0.25
technical   技术面   0.22
capital     资金面   0.18
```

- 用环境变量 `W_NEWS / W_SENTIMENT / W_FUND / W_TECH / W_CAPITAL` 覆盖；脚本自动归一化。
- **市场降级再分配**：美/日/韩无资金流时，`capital` 权重按比例摊给其余四因子（见报告里
  `weights_used`，capital=0）。
- **风格自适应**（Claude 可在报告里说明并临时调权）：
  - 题材/事件驱动（消息为王）：上调 `news`、`sentiment`，下调 `fundamental`。
  - 价值/长线持有：上调 `fundamental`，下调 `news`、`technical`。
  - 短线波段：上调 `technical`、`capital`。

## 3. 综合分 → 操作映射

`composite = Σ (因子分 × 权重)`，0–100：

| 综合分 | 结论 | 立场 |
|--------|------|------|
| ≥ 68 | 买入 / 增持 | 偏多 |
| 56–68 | 逢低买入 / 持有偏多 | 中性偏多 |
| 45–56 | 持有 / 观望 | 中性 |
| 33–45 | 减持 / 逢高减仓 | 中性偏空 |
| < 33 | 卖出 / 回避 | 偏空 |

## 4. 风险否决（override，优先于综合分）

无论综合分多高，命中以下任一，结论至少下调一档并在"风险警报"中红字标注：

- **基本面雷**：财务造假/退市风险/商誉爆雷/连续亏损 + 现金流为负。
- **消息黑天鹅**：立案调查、实控人被查、核心客户/订单流失、重大诉讼。
- **政策反转**：行业被点名整顿、补贴退坡、加征关税/出口管制直接命中主营。
- **技术破位**：跌破关键支撑(年线/重要平台)且放量、周线死叉。
- **拥挤交易**：情绪亢奋(温度>85)+ 获利盘比例>90% + 主力连续净流出 → 高位风险。
- **流动性**：成交极度萎缩或停牌风险。

## 5. 五因子打分口径速查

**技术面**（`technical_score`）：均线多/空头排列 ±12、MACD 金/死叉 ±8、
RSI 超买-7/超卖+7/健康+3、KDJ J 高低 ±4、站上/跌破 MA20 ±5、20 日动量 ±8。

**资金面**（`capital_score`，仅 A 股）：主力连续净流入 +(天数×4，封顶+18)，
连续净流出对称扣分；结合筹码获利比例与龙虎榜。

**基本面**（P0 语义）：净利同比 >30% +12 / 正增长 +5 / 下滑 -12；营收同比、ROE、
毛利率、PE 区间加减分。grade 强(≥65)/中(45-64)/弱(<45)。

**情绪面**：A 股=上涨家数占比基底(30-80)+涨跌停差+VIX(30%)+大盘趋势；
其它市场以 VIX 反向映射为主。标签：极度恐慌/恐慌/中性/乐观/亢奋。

**消息面**：见 `prompts.md` 的"拆解预言"四步，由 Claude 给 0–100 并写明依据。

## 6. 邻近标的比选（第三块要求）

对 1–3 只同行业可比公司，用同一套五因子打分，列对比表（估值 PE/PB/PS、增速、
ROE、资金面、消息催化、综合分），回答："同样的钱，为什么买这只而不是那只？"
比选维度：**估值性价比、增速弹性、确定性(验证程度)、催化临近度、拥挤度**。

## 7. "弹性 / 验证"链条（借鉴买方研报 memo）

每条核心逻辑都走： **新闻/需求变化 → 财报传导 → 业绩弹性 → 验证路径**。
- 弹性：该催化对 EPS/估值的量级影响（拍量级，标注假设）。
- 验证：用什么数据、在什么时点能证实/证伪（下季财报、月度数据、订单公告）。
未被验证的逻辑标 ⏳，已证实标 ✅，证伪标 ❌——只有 ✅/高弹性才显著加分。
