# company-profile/ —— 本单位资料参考文件夹

把**本设计单位的资质、业绩、人员、奖项、关系、财务**资料放进这个文件夹。技能在复盘/决策时会**先读这里**，再逐项对照本次招标条件，给出"对照式"结论（能不能过硬卡口、哪个分项是洼地、若参标排第几），而不是泛泛而谈。

## 怎么用
1. 复制 `company-profile-template.md`，按实际填写（或直接编辑它）。
2. 业绩库、奖项库数据多的，可另存为 `业绩库.csv` / `奖项库.csv` / Excel 放进本文件夹，技能会一并读取。
3. 资质证书、社保截图等扫描件也可放入备查（技能主要读结构化清单）。
4. 信息更新后只需改这里，下次复盘自动用最新画像。

> 隐私提示：本文件夹含公司敏感资料。若不想随技能推送到公开仓库，把本文件夹加入 `.gitignore`，只在本地保留。模板（不含真实数据）可保留在仓库。

## 技能读取的关键字段
- 持有资质（类别+等级+有效期+范围）
- 业绩库（项目/业主/类型/单项面积/合同额/是否纯设计专项/施工图审查合格日期/竣工/获奖）
- 关键人员（一级注册建筑师、职称、社保归属、主持业绩、年限）
- 奖项库（颁奖单位/奖项/等级/年份——区分建筑学会奖 vs 中国建筑装饰协会奖）
- 团队职称储备（建筑高工、机电高工人数）
- 关系/属地基础（业主系统、城市合作史与入口）
- 财务（保证金/履约担保资金占用承受力、毛利率假设）
