# 解析几何剖面法 · Analytic-Geometric Explication

> 庖丁解牛，"依乎天理，批大郤，导大窾"——不是乱砍，而是顺着事物**内在的纹理**去剖。
> 这套方法就是把一个原理**当作一具有内在结构的几何本体**，再用解析几何顺着它的"理"去剖、去照、去展开，让逻辑自己显形。

## 哲学内核（先读这一段，它决定一切）

每一个原理都有两层：
- **本体（理 / 不变量）**：那个生成一切表象的、不变的内在结构。在年轮例子里，是"圆心向外、疏密有律的同心环"。
- **显象（相 / 表象）**：本体在某个**切面**上呈现出来的样子。**同一具年轮**——
  - 顺着轴心剖（**径切**）→ 是一组**平行直线**；
  - 偏离圆心剖（**弦切**）→ 是一族**嵌套抛物线**（山形纹 / 大教堂纹）；
  - 斜着剖（**斜切**）→ 是一圈圈**被拉伸的椭圆**。

**显象的差异，不来自本体的差异，而来自"切法"的差异。** 这就是这套图最迷人的地方：它用一条条可推导的曲线，把"本质不变、显象随观照角度而变"这件事，**变成了眼睛能看见的东西**。

> 所以这套方法的任务永远是回答三个问题：
> **本体是什么（理）？ 我们用什么去剖它（切法）？ 每一种剖法解析地导出什么曲线（显象）？**
> 而真正的智力诚实在于——**每一条线都从一个方程来，而不是手画的"看起来像"**。

## 何时使用

- 用户给出一个原理 / 关系 / 结构，想要"严谨而有美感地画出来"，而不是配图示意。
- 需要展示"同一个东西，从不同角度/切法/参数看，呈现不同形态"的统一性。
- 想把自然规律、数学关系、工程构造**规整化**成可标注、可建造的几何造型板。
- 用户喜欢年轮三切法（V3 系列）那种"解析几何意境图"，想复制这套表达。

## 七步流程（这就是被蒸馏的方法）

按顺序做。每一步都对应原 V3 代码里的一种手法，已在 `scripts/explicate.py` 里做成可复用积木。

### 1 · 本体建模（Model the essence）
把原理还原成**生成式参数模型**——一个由少数参数控制、能长出全部表象的几何对象。
- 找到"不变结构"：圆柱+同心环 / 一个曲面 / 一个高维体 / 一族等值线。
- **韵律而非随机**：表象的疏密、起伏用**叠加正弦**生成（`ring_radii()`），得到"有机却确定"的节奏，而不是 `random`。这是这套图"活"的来源。

### 2 · 切法定义（Define the operators of revelation）
列举"显露本体"的方式，每一种都写成**解析对象**：
- **平面切**：点 + 法向（径切 = 过轴面 `y=0`；弦切 = 偏移面 `y=y₀`；斜切 = 倾斜面 `z=a+bx+cy`）。
- **投影**：把高维本体投到不同平面（同一立体投出圆/方/三角的"影"）。
- **参数扫描**：固定一条方程，扫一个参数（离心率变化 → 圆/椭圆/抛物线/双曲线一族圆锥曲线）。
- **变换**：同一结构在连续变换下的相位。

### 3 · 解析求交（Derive the manifestation in closed form）— **核心**
对每一种切法，**闭式求解**交线 / 投影 / 等值线，让显象是"算出来的"，逻辑透明：
- 同心圆 ∩ 过心面 → 平行线（环被侧看成线，疏密=半径差）。
- 半径 r 的圆 ∩ 面 `y=y₀` → `x=±√(r²−y₀²)`，在板上展开为抛物线 `z=z₀+A(1−(x/x_c)²)` → 嵌套山形纹。
- 同心圆 ∩ 斜面(角 θ) → 椭圆，长轴被 `1/cosθ` 拉伸。
> 详见 `references/math-derivations.md`。**这一步不许偷懒**：宁可换简单的本体，也要保证曲线来自方程。

### 4 · 多视图编排（Coordinate the views）
同一真理，用 3–5 张协调的视图回答不同认知问题：
| 视图 | 回答的问题 | 原 V3 对应 |
|---|---|---|
| **在体定位** | 切面在本体的哪里、什么角度？ | 图1 树干+三切面 |
| **单面纹理** | 每个切面上各显什么象？ | 图2 三面纹理立体 |
| **爆炸对应** | 切法 ↔ 纹理 如何一一对应？ | 图3 爆炸视图（连线） |
| **规整标注** | 如何规整成可造、可量的造型？ | 图4 几何造型板+尺寸 |
| **多角度旋转** | 它的三维形态到底如何？ | 图5 旋转视图矩阵 |

### 5 · 规整化（Regularize：从自然律到人造物）
把解析导出的"自然曲线"映射到**标准化、可建造、带尺寸标注**的面板上。哲学上：把天之理，化为人之器。保留疏密韵律，去掉偶然噪声。

### 6 · 审美编码（Aesthetic DNA）
严格遵守 `references/aesthetic-system.md`：暖奶油底（`#faf5ee`/`#fff5f0`）、每个切法族一个克制的主色、细暖线、半透明叠面、**线宽随疏密变化**（`lw ∝ spacing`）、极轻噪声扰动赋予生命、`axis('off')`、大量留白。气质是**科学的诗**——精确而温润。

### 7 · 导出（Export）
高清 PNG：`dpi=200, bbox_inches='tight', facecolor='white'`；中文字体 `Microsoft YaHei`/`SimHei`；一张图一个认知视图。

## 怎么运行

复用引擎在 `scripts/explicate.py`（积木：韵律半径、圆柱、平面求交、椭圆切、审美常量、保存器）。

1. **照搬复刻**：直接运行参考实现，产出年轮 V3 五图——
   ```bash
   python scripts/example_tree_rings.py --out "C:\path\to\output"
   ```
2. **解析新原理**：与用户确认 ①本体是什么 ②用哪些切法/投影/参数 ③每种切法解析导出什么曲线；然后 `from explicate import *`，照七步搭一份新脚本（以 `example_tree_rings.py` 为骨架）。先在 `references/methodology.md` 里挑一个**原型**（剖面型/投影型/参数扫描型/变换型）匹配用户的原理。
3. **审美校验**：出图后对照 `aesthetic-system.md` 自检；线条是否都来自方程（智力诚实自检）；多视图是否覆盖"何处/何象/何以对应/如何造/三维何形"。

## 关键约束（智力诚实）

- **每一条曲线必须来自一个方程**，不许手画"形似"。看不出闭式解时，换一个能解的本体，而不是假装。
- **不变量要真不变**：所有显象必须由同一组本体参数生成；改本体则所有视图同步变。
- **疏密用正弦叠加，不用随机**；噪声只作极轻微的"生命扰动"。
- 若某切法在某半径无交线（如弦切处 `r<y₀`），**如实跳过**，不要补线。
