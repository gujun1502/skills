# 既有建筑改造：现行规范 vs 三十年前原图 — 缺项清单

> 案例底本：中国银行奉化支行大楼解危加固外立面维修及配套工程
> 适用：1990 年代或更早建造的中高层办公/银行/商业类既有建筑，需做"整体改造（非修缮）"认定，按现行规范走施工图审查。

---

## 0. 前置定性问题（最贵的一项缺项）

| 项目 | 现行规范要求 | 三十年前原图状况 | 缺项性质 |
|---|---|---|---|
| 项目性质界定 | 审图机构会按地方《既有建筑改造消防/抗震技术指南》判定为"整体改造"或"修缮"，前者必须全部按现行规范补齐 | 原始立项常含糊为"维修加固" | **决定全盘标准** — 一旦定性"整体改造"，下面所有缺项都必须补 |
| 原消防验收合格证明 | 必须提供（或官方认可的等效证明） | 当年文件多已散佚，且当年验收标准低于现行 | 无可替代，须业主到档案馆/消防大队调取 |
| 原始审图章 | 现行图纸均需审图章 | 多为手绘、无审图章 | 原图作废，只能作"现状勘测参考"使用 |

## 1. 建筑专业 — 缺项

### 1.1 资料类
- **可行性研究报告**（含改造定性、规模、投资）
- **既有建筑改造消防技术可行性研究报告**（地方指南强制，第三方资质单位出具）
- **节能诊断报告**（第三方检测机构，节能与幕墙专项报批的前置）
- **原建筑测绘图**（含违章搭建的真实现状，不可直接用原图）
- **市政重点保护区评估报告**（如涉水源/历史/生态控制线）

### 1.2 图纸/计算类
- **现状勘测图**（含全部既有违章搭建，外墙、屋面、夹层、空调外机位）
- **节能专项设计**（外墙、屋面、门窗 K 值，热工计算书）
- **幕墙专项设计 + 幕墙计算书**（结构、热工、防火、防雷四套）
- **无障碍设计专项**（出入口、卫生间、轮椅回转、盲道）
- **防火分区/疏散计算书**（按现行《建规》《高规》重新核算）
- **楼梯前室/合用前室面积复核**（很多 90 年代楼梯间无前室或前室不达标）
- **自然排烟窗有效面积计算**（按当层防烟分区面积复核）

### 1.3 实物现状缺项（设计前必须勘察）
- 屋面违章构筑物清单与拆改方案
- 原外立面构造做法（保温层/防水层是否存在）
- 室内层高净高复核（含原暗梁、原管线占用）

## 2. 结构专业 — 缺项

### 2.1 资料类（最易被卡）
- **法律效力的第三方《地质勘察报告》**
  - 若引用周边老地勘，需同时提供官方证明：参考勘察点距本工程 ≤50 m 且地质条件相似
- **安全性鉴定报告**（含主楼、附楼分别出具）
- **抗震鉴定报告**（按现行《建筑抗震鉴定标准》GB 50023）
- **原始地基承载力数据**（90 年代常缺，必须重新静载或动探补做）
- **原结构计算书**（多为手算手稿，与现场不符）

### 2.2 图纸/计算类
- **现状结构测绘图**（梁高、板厚、柱截面、墙厚必须现场实测，原图不可信）
  - 案例教训：原图按 100 mm 板厚设计开槽走线，**实测为 60 mm**，只能小型工具凿除
- **加固设计专项**（粘钢、碳纤维、外包钢、混凝土加大截面等）
- **新增机房荷载复核**（消防水池、生活水箱、屋顶外机、新增机房均需复核相邻结构）
- **桩基方案适配现场施工条件**
  - 案例教训：原设计预应力管桩，现场南侧无打桩场地，改为混凝土锚杆静压桩
- **拆除方案与机械参数验算书**（多机同层、机械重量、振动影响）
- **隐蔽工程探查报告**（原地下管网、电缆敷设走向）

### 2.3 实物现状缺项
- 原承重墙/剪力墙位置（设计能否开槽、能否走桥架全靠这个）
- 原基础形式、埋深、桩长（无原图时需局部开挖核实）
- 原结构裂缝、变形、腐蚀普查

## 3. 给排水专业 — 缺项

### 3.1 系统级缺项（现行规范强制，原建筑往往全无）
| 系统 | 现行要求 | 原建筑常见状况 |
|---|---|---|
| 消防水池 | 容积按现行《消水规》GB 50974 核算 | 容量不足或无独立泵房（案例：缺 252 t 池 + 133 m² 水池间 + 85 m² 泵房） |
| 屋顶高位消防水箱 | 最小有效容积 + 稳压泵 | 无稳压泵，水箱过小（案例需 18 t） |
| 室外消防水泵接合器 | 数量、间距按现行规范 | 不合规或缺失 |
| 生活/消防供水分离 | 严格分质分系统 | 混用 |
| 分区供水 | 高区需变频/水箱 | 市政直供 1–8F，3F 以上压力不足 |
| 雨污分流 | 强制 | 雨污混流 + 化粪池容量不足 |
| 屋面排水 | 重现期、雨水斗复核 | 积水、防水失效、地基渗水 |
| 化粪池 | 按人数核算 | 容量不足，与市政驳接不通 |

### 3.2 图纸/计算类
- 室内外给水、消火栓、喷淋系统图与平面图
- 雨水、污水、废水系统图（分流）
- **消防水力计算书**（最不利点压力、流量）
- 生活变频泵选型与水箱间结构荷载
- **管网综合**（外场新增大荷载机房不得影响相邻建筑）
- 防水专项（地基外墙、屋面、卫生间、机房）

## 4. 暖通专业 — 缺项

### 4.1 系统级缺项
| 系统 | 现行要求 | 原建筑常见状况 |
|---|---|---|
| 新风系统 | 公建必设，按 GB 50736 与节能规范 | 完全无新风（案例补全热交换器，每层吊装 + 外墙百叶取风） |
| 楼梯间防烟 | 自然通风不达标时需机械加压送风 | 仅自然通风，不满足现行《建筑防烟排烟系统技术标准》GB 51251 |
| 排烟系统 | 自然排烟有效窗面积按防烟分区核算；不足需机械排烟 | 自然排烟面积不足 |
| 空调系统统一性 | 屋顶机位排布、外立面景观 | 分体机外挂混乱 |
| 室外机荷载 | 屋面新增荷载需结构复核 | 原结构未预留 |

### 4.2 图纸/计算类
- **加压送风/排烟计算书**（风量、余压、风口面积）
- **新风量与冷热负荷计算书**（按现行节能规范）
- 新增加压机房、排烟机房选址与结构加固
- 外墙百叶位置 vs 立面/幕墙协调
- **设备振动、噪声对相邻房间的评估**（机房选址下移到二层时必做）
- 与电气、给排水的剖面/管井综合

---

## 5. 电气专业 — 缺项

### 5.1 系统级缺项
| 系统 | 现行要求 | 原建筑常见状况 |
|---|---|---|
| 消防电源 | 双电源末端切换，专用回路 | 消防电与非消防电混线 |
| 火灾自动报警 | 强制全覆盖 | 缺失或形同虚设 |
| 应急照明与疏散指示 | GB 51309 集中电源/集中控制型 | 照度/持续时间不达标 |
| 防火门监控 | 强制 | 无 |
| 消防联动 | 全联动 | 无 |
| 漏电保护 RCD | 人员可达插座回路强制 | 多数老回路无 RCD |
| 高压配电室容量 | 按改造后总负荷扩容 | 电量不够且无扩容空间 |
| 防雷接地 | 接地电阻复测 + 新增金属构件接闪带连通 | 接地体锈蚀超标，新增设备未接入 |
| 配电箱/电表间 | 不占疏散通道、IP 等级、操作距离 | 占通道、箱体锈蚀 |
| 应急照明（公区） | 楼梯/走廊 GB 51309 | 照度/续航不足 |

### 5.2 图纸/计算类
- **负荷计算书**（按改造后实际负荷，含新增多联机、变频泵、机房）
- **短路电流计算 + 选择性整定**
- 高压系统主接线、低压配电系统图、平面图
- 火灾自动报警系统图与平面图
- 应急照明系统图与平面图
- 防雷接地平面与详图
- 弱电（综合布线、视频监控、门禁）
- **桥架/竖向干线路由**（与结构开洞、暖通管井协调）

### 5.3 实物现状必查项
- 原电缆绝缘老化测试、开关额定电流匹配、接头氧化腐蚀、箱体锈蚀
- 近 1–3 年电表数据（无则钳形表实测各回路电流）
- 原管线走向（明装可保留，暗装除非检测安全否则全部重敷）

## 6. 跨专业 / 流程类缺项

- **造价拆分导致的设备闭环问题**：项目拆为"室内装修"+"室外配套"两个标段时，给排水、消防、电气往往跨标段，导致单一标段图纸无法形成闭环、审图与验收均卡壳。必须在立项阶段统筹设备专业范围。
- **分阶段实施 vs 审图整体性冲突**：一阶段只报结构加固+室外管网，但审图会要求一并提交二阶段（建筑+设备）图纸以判定改造规模，分阶段策略需提前与审图机构沟通确认。
- **副楼范围与核心机房选址的死结**：副楼若不在立项范围内，其内的低压配电室、UPS、消防/生活泵房必须迁入主楼一层，需业主先决策范围再开展设计。
- **市政接口资料**：电力、给水接入点位置/容量/标高、场地标高、周边管线管廊 — 全部需业主侧协调市政部门提供。
- **市政消防能力**：若市政消防水源/消防车道不满足改造后需求，需同步启动市政改造工程。

## 7. "拿到原图"也不能直接用的三条铁律

1. **手绘无审图章 = 仅作勘测参考，不可作为设计依据**
2. **现场实测优先于原图**（板厚、梁高、墙位、管线走向、设备容量均以实测为准）
3. **隐蔽工程必须探查后再设计**（地下管网、电缆走向、原基础形式拆开前不出最终图）
