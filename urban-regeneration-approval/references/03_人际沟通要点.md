# 既有建筑改造 — 人际沟通要点

> 四方关系：业主（出资+决策）/ 审图机构（合规裁决）/ 总包+施工队（落地）/ 设计院（你自己）。
> 任何一方信息不对称，最后都会以变更、超期、超造价的方式回到设计院头上。

---

## 一、与业主的沟通

### 1.1 立场与边界
- **设计院 ≠ 业主代理人**：法律责任在设计资质上，不替业主做"豁免规范"的决定。
- 凡是"我们就不做了吧""能不能省一点"——必须以**正式发函+造价/合规影响说明**回复，留痕。
- 案例：奉化项目通过《情况说明函》把"超出设计院职责范畴、需业主协调解决"的事项逐条列给中行领导（第三方报告/历史资料/市政证明/副楼决策），把球清晰地推回业主，并保留底稿。

### 1.2 表达方式
- 面对**非技术背景的业主领导**：用"如果不解决会怎样 / 谁来出钱 / 多少时间"三段式，不要堆术语。
- 面对**业主项目经理**：用对照表 + 责任矩阵，告诉他"你方第几条没给，我方第几张图无法出"。
- **不要私下口头承诺**：所有承诺以邮件/函件确认，回复时附"已收阅"。

### 1.3 关键时刻的话术模板
- "依据 XX 规范第 X.X 条，本项目不满足现行要求；如继续按原状执行，存在 XX 法律/安全风险，需业主书面确认。"
- "本意见超出我方设计职责范畴，需业主方协调 XX 方解决，否则我方设计无法推进，时间影响约 X 周。"
- "若按现行规范全面整改，预计造价增加约 XX，请贵方决策。"

### 1.4 雷区
- 不要替业主向审图机构"解释" — 业主不在场的解释审图机构不认。
- 不要"先做了再说" — 业主默认是"反正你设计院兜底"。
- 不要在群里讨论关键决策 — 群消息撤回快，无法作为责任凭证。

## 二、与审图机构的沟通

### 2.1 关键认知
- 审图机构**只对图纸是否符合现行规范负责**，不对业主造价、工期负责。
- 一旦正式审图意见下发，回旋空间极小；**预沟通**才是真正能改变结果的环节。

### 2.2 预沟通要点（报建前做）
- **性质判定预沟通**：把改造内容、范围、立项文件提前递审图机构，争取"修缮"性质判定。
- **分阶段策略预沟通**：能否一阶段（结构加固+室外管网）单独报审，二阶段后补。
- **降标条款预沟通**：原建筑无扩容条件的（如高压配电室容量、楼梯前室面积），询问是否接受替代方案。

### 2.3 正式审图沟通
- 审图意见单**逐条书面回复**，不要电话答复后忘记留痕。
- 涉及业主决策的，先发函业主拿到书面确认再回复审图。
- 沟通时**记录沟通对象姓名+日期+口径**，必要时让审图机构邮件确认。

### 2.4 雷区
- 不要绕过审图找上一级"通融" — 一旦被记住，后续所有项目会被严查。
- 不要"少画一点蒙混" — 审图会要求补出二阶段图纸，反而暴露前端不合规。

## 三、与总包/施工队的沟通

### 3.1 关键认知
- 施工队是**现场真实情况的第一发现人**，他们的反馈往往比测绘报告更准。
- 他们也是**变更签证的最大动力来源**，会主动找设计院要变更。

### 3.2 关键节点
- **进场首周现场复核会**：所有专业到场，把原图与现场偏差一次性列清，三方签字。
- **拆除前方案评审**：机械参数、振动影响、楼板验算、扬尘控制，**让结构出具拆除验算意见**。
- **隐蔽工程探查后再出正式图**：地下管网、电缆、原基础形式以现场为准。

### 3.3 施工队的"善意建议"要分两类
- 类型 A：**有利于施工、不影响合规** —— 鼓励采纳并签证（如奉化把预应力管桩改为锚杆静压桩）。
- 类型 B：**省事但绕开规范** —— 必须拒绝，并书面回复"按图施工"。
- 案例：奉化施工队建议"原电缆能用就别拆"——设计院拒绝，理由是绝缘老化与防火回路无法核实，写函留痕。

### 3.4 雷区
- 不要在现场口头改图 — 全部以工程联系单/设计变更单为准。
- 不要默许"先施工后补图" — 出问题时设计院承担连带责任。

## 四、设计院内部协同

### 4.1 跨专业协同
- 建筑+结构+设备三专业的**最早一次综合会议**，应该在拿到第三方报告之后立刻召开（不是初设阶段才开）。
- 设备 → 结构传递荷载点的时间窗口固定（如每周三例会），避免"最后一刻才发现机房没地方"。
- **剖面综合**：核心筒、走廊、机房通道在初设阶段就出剖面，全专业签字。

### 4.2 与幕墙/节能/咨询单位
- 幕墙单位常常**独立向审图机构对接**，可能与建筑院方案不一致 → 设计院必须把幕墙也纳入总控。
- 节能诊断报告**不要让建筑院自己出**（自己出自己审，审图机构不认）。

---

## 五、沟通记录的归档纪律

| 沟通类型 | 必留痕载体 | 保存位置 |
|---|---|---|
| 业主决策 | 正式函件 + 业主签字回函 | 项目档案 + 设计院档案 |
| 审图沟通 | 邮件/书面纪要 | 报建档案 |
| 施工偏差 | 工程联系单/设计变更单 | 施工档案 |
| 跨专业内部协同 | 会议纪要 + 签字 | 设计院档案 |
| 紧急情况口头沟通 | 24h 内补发邮件确认 | 邮箱 |

> 一句话：**没有留痕的沟通等于没有发生过**。
