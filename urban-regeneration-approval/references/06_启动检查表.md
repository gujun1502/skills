# 项目启动检查表（既有建筑改造）

> 用法：每个新项目在合同签订后 48 小时内逐项核对，每项必须有"完成 / 在办 / 待业主"三态之一。

---

## A. 基础信息

- [ ] 项目所在城市/区
- [ ] 原建筑建成年代、原图比例尺、是否手绘、是否有审图章
- [ ] 业主主体性质（央企/国企/民企/政府）
- [ ] 总投资估算 + 资金到位时间
- [ ] 期望工期 + 关键节点（如银行支行的"对外营业窗口"）

## B. 项目定性

- [ ] 立项文件中的项目性质表述（修缮 / 改造 / 整体改造）
- [ ] 与审图机构预沟通的性质判定（书面）
- [ ] 与消防大队预沟通的口径

## C. 本地政策清单

- [ ] 本地《既有建筑改造消防技术指南》版本与发布日期
- [ ] 本地《抗震加固技术规程》版本
- [ ] 是否涉及历史风貌/文保保护区
- [ ] 节能气候分区与最新执行标准
- [ ] 是否享受政策优待/快速通道

## D. 业主必交资料（业主资料移交清单）

- [ ] 原产权与历史档案
- [ ] 原消防验收合格证（或等效文件）
- [ ] 原全套图纸 + 原审图意见
- [ ] 原结构竣工验收文件
- [ ] 市政电力接入点（位置/容量/标高）
- [ ] 市政给水接入点（位置/压力/容量）
- [ ] 市政雨污接入点 + 市政管线管廊资料
- [ ] 市政燃气接入资料
- [ ] 周边场地标高
- [ ] 市政消防资料（水源、消防车道）
- [ ] 重点保护区评估（如适用）

## E. 三份强制第三方报告（业主侧招标启动）

- [ ] 既有建筑改造消防技术可行性研究报告
- [ ] 节能诊断报告
- [ ] 合规的地质勘察报告（或合规引用证明）

## F. 结构鉴定

- [ ] 安全性鉴定报告 — 主楼
- [ ] 安全性鉴定报告 — 附楼
- [ ] 抗震鉴定报告 — 主楼
- [ ] 抗震鉴定报告 — 附楼

## G. 现状勘察

- [ ] 现状测绘报告（含违章搭建）
- [ ] 隐蔽工程探查计划 + 拆除前进度
- [ ] 原结构裂缝/变形/腐蚀普查
- [ ] 原电缆绝缘老化测试 + 近 3 年电表数据
- [ ] 原给排水管道与化粪池状况

## H. 设计输入决策点（业主签字）

- [ ] 副楼是否纳入设计范围
- [ ] 项目拆分边界（室内/室外/外立面/管网）
- [ ] 设备机房选址首选方案
- [ ] 必保留景观/树木/历史构件清单
- [ ] 分阶段报审策略（一阶段+二阶段 / 整体）

## I. 报审策略

- [ ] 报建主管部门确认
- [ ] 审图机构选定与预沟通会议
- [ ] 消防大队预沟通会议
- [ ] 节能/幕墙专项是否并联
- [ ] 文保意见（如适用）

## J. 设计院内部

- [ ] 项目负责人 + 各专业负责人指定
- [ ] 跨专业首次综合会日程
- [ ] 每周例会机制
- [ ] 与幕墙/节能/咨询单位的协同机制
- [ ] 与总包/施工队的进场首周复核会日程

---

## K. 红线项（任何一项 = 不可启动设计）

- [ ] 项目性质判定有书面文件
- [ ] 三份第三方报告已招标启动
- [ ] 业主资料移交清单已签字
- [ ] 副楼范围已书面决策
- [ ] 现状测绘已启动
