# 交易机制速查：竞价规则 · 尾盘资金 · 红绿黑成交口诀

供技术面与资金面解读时引用。**口诀是经验法则，非必然规律，须与基本面/消息面交叉验证。**

---

## 1. A 股竞价与涨跌停

- **集合竞价**：9:15–9:25 开盘集合竞价（9:15–9:20 可撤单，9:20–9:25 不可撤）；
  14:57–15:00 收盘集合竞价（科创板/创业板/北交所）。
- **连续竞价**：9:30–11:30、13:00–14:57。
- **涨跌幅**：主板 ±10%，创业板/科创板 ±20%，北交所 ±30%，ST ±5%。
- **看竞价**：高开放量、竞价阶段量能突变、9:25 未匹配大单 → 当日情绪/资金意图先行信号。
  - 高开+竞价持续放量承接 → 强势；高开但竞价量缩、9:25 砸盘 → 诱多嫌疑。

## 2. 美股竞价与机制

- **盘前** 4:00–9:30 ET，**盘中** 9:30–16:00 ET，**盘后** 16:00–20:00 ET。
- 开盘集合竞价(NYSE/Nasdaq Opening Cross)、收盘竞价(Closing Cross)；
  收盘竞价成交量极大（指数调仓、被动资金），尾盘方向参考价值高。
- 无统一涨跌停，有**熔断**（标普 7%/13%/20%）与个股 LULD 波动限制。
- 财报多在盘后/盘前发布 → 关注 after-hours/pre-market 跳空。

## 3. 尾盘资金 / 主力意图（对应需求"尾盘资金流入走势、前一天尾盘"）

脚本 `fetch_capital_flow.py` 用**日度主力净额序列**近似主力意图，给出
`direction`(净流入/净流出) 与 `streak_days`(连续天数)。解读：

- **尾盘拉升+主力净流入** → 次日高开预期，资金做多意愿强（但警惕尾盘对倒拉高出货）。
- **尾盘跳水+主力净流出** → 护盘失败/出货，次日低开概率大。
- **前一日尾盘净流入、今日延续** → 资金连续性好，趋势更可信（streak 越长越强）。
- **量价背离**：股价新高但主力净流出（红绿黑里的"红肥绿瘦却暗流出"）→ 顶背离预警。
- A 股可叠加：北向资金、龙虎榜席位（游资/机构）、筹码获利比例(>90% 警惕)。

> 分时级"尾盘最后半小时净流入"需 Level-2 分时资金数据；当前用日度近似，
> 报告中应注明这一口径局限，避免过度解读。

## 4. 红绿黑成交分析口诀（量价情绪统计）

把成交按方向/性质着色，统计强弱。**红=主动买盘(外盘)，绿=主动卖盘(内盘)，
黑=中性/大宗对倒**。常用研判：

- **红肥绿瘦**：外盘显著大于内盘、价升量增 → 多头主动，健康上涨。
- **绿肥红瘦**：内盘大于外盘、价跌量增 → 空头主动，承压。
- **红肥但价滞涨**：高位放量滞涨、外盘虚高 → 对倒诱多/出货嫌疑（黑单增多佐证）。
- **绿肥但价不跌**：低位放量价稳、内盘虚高 → 主力吸筹/洗盘嫌疑。
- **黑单骤增**：大宗/对倒比例升高 → 主力调仓，方向需结合次日确认。
- **缩量十字**：多空平衡、观望，等方向选择。

口诀总纲：**"量在价先，红绿看主动，黑单辨真伪；背离需警惕，连续定趋势。"**

## 5. 与因子分的衔接

- 红绿黑/尾盘资金 → 主要影响 `technical`(量价) 与 `capital`(主力) 子分的**人工校正**。
- 当脚本量价信号与红绿黑口诀冲突时，在报告"决策过程"里写明分歧与你的取舍权重。
