# 早盘决策框架（premarket-framework）

本文件说明早盘 skill 的**预测口径与决策逻辑**，确保结论可追溯、可调、可复盘。
配合 `premarket.py` 的输出阅读。

---

## 一、五因子与权重

| 因子 | key | 默认权重 | 含义 | 数据来源 |
|------|-----|---------|------|---------|
| 技术面 | technical | 0.25 | 个股量价结构/动量(均线/MACD/RSI/KDJ/BOLL/动量) | `fetch_market.py` |
| 资金面 | capital | 0.22 | 主力(超大单/大单)前一日及连续净流入流出、小单承接 | `fetch_capital_flow.py` |
| 宏观隔夜 | overnight | 0.20 | 美股隔夜 + 亚太早盘(日韩已开盘) + 汇率利率 + 个股隔夜代理 | `fetch_overnight.py` |
| 板块联动 | linkage | 0.18 | 同业/行业ETF前一日合力 + 信息面主题 | `fetch_linkage.py` + `linkage.json` |
| 情绪面 | sentiment | 0.15 | 市场情绪温度 | `sentiment.py` |

- 权重可用环境变量覆盖：`W_PM_TECH / W_PM_CAPITAL / W_PM_OVERNIGHT / W_PM_LINKAGE / W_PM_SENTIMENT`，写在 `config.env`，自动归一化。
- **市场降级**：非 A 股没有资金流/板块联动 → 该因子置 50 且权重按比例摊给其余因子（`weights_used` 体现）。
- **早盘分映射**：≥66 积极参与/低吸｜56–66 谨慎偏多/逢回踩｜45–56 观望不追高｜34–45 防守/反弹减仓｜<34 回避/反弹了结。

## 二、今日高低点预测（9:30–15:00）

分三步，全部写进 `prediction.today`：

1. **预测开盘缺口** `gap_pct`
   - 由"外部偏向" `ext_bias` 主导（= 隔夜 overnight + 板块联动 linkage 的均值映射到 −1~+1）。
   - `gap_pct = clamp(ext_bias × 1.5, −3%, +3%)`；`预测开盘 = 昨收 ×(1+gap)`。
   - 逻辑：跳空主要由隔夜外盘与板块前一日强弱决定，个股自身技术面影响盘中而非跳空。

2. **日内波动幅度** = `ATR14 ×(0.9 + 0.4×|bias|)`
   - `bias` = 综合早盘分映射到 −1~+1。趋势越强(|bias|大)，预期振幅越大。
   - 多空劈分：`上行占比 = clamp(0.5 + 0.35×bias, 0.2, 0.8)`，下行占比 = 1 − 上行。
   - `预测高点 = 预测开盘 + 幅度×上行占比`；`预测低点 = 预测开盘 − 幅度×下行占比`。

3. **技术位校准**（理论价位）
   - 高点：若非强多(bias<0.5)，高点不轻易穿越**最近压力位**(`high_ref = min(预测高, 最近压力×1.005)`)。
   - 低点：若非强空(bias>−0.5)，低点获**最近支撑位**托底(`low_ref = max(预测低, 最近支撑×0.995)`)。
   - 支撑/压力来自 `technical_levels`：MA20/60/120、BOLL 上下轨、年内高低。

> 输出 `predicted_high/low`(ATR 模型) 与 `high_ref/low_ref`(技术位校准后) 两套，互为印证。
> **这是概率区间，不是精确点位**；报告里必须这样表述。

## 三、未来三天趋势与理论高低位

- **趋势偏向** `trend_bias` 用慢变量：技术面 + 资金面 + 板块联动 三者均值映射到 −1~+1（情绪/隔夜对三日影响衰减，故不计入）。
- **三日幅度** = `ATR14 ×√3 ×(0.9 + 0.4×|trend_bias|)`（√3 来自波动随时间的近似扩张）。
- **理论高低位**：向中级别结构靠拢——
  - 偏多(trend_bias>0.2)且上方有 MA60/MA120/年内高在可达范围 → 取最近的中级别压力为理论高点。
  - 偏空(trend_bias<−0.2)且下方有中级别支撑 → 取最近支撑为理论低点。
- **趋势标签**：>0.35 偏强上行｜>0.12 震荡偏多｜>−0.12 震荡整理｜>−0.35 震荡偏弱｜否则 偏弱下行。
- **波浪/位置提示** `wave_position`：用年内位置(pos_in_range_pct)给出"高位防回落 / 中继需放量 / 低位弹性大"的定性提示——**精确数浪交给 Claude 结合形态判断**，脚本只给位置锚。

## 四、Claude 要做的语义层（脚本不做）

脚本给的是确定性数字，**以下由 Claude 在报告里完成**：

1. **信息面解读**：读 `blocks.overnight`(隔夜板块/个股代理)、`blocks.linkage.themes`(苹果链/先进封装/存储/A50…)、`blocks.news`(若开 --news)，判断"今天有没有针对该股/板块的催化或利空"，据此**手动微调**预测区间与建议（说明理由）。
2. **盘口解读**：读 `blocks.orderbook.signals`(主动买盘 active_buy_ratio、挂单委比、量能)，校正当日强弱判断。
3. **风险否决**：命中以下任一，**下调结论并红字警报**——
   - 隔夜对应板块大跌(代理 chg < −3%) / 人民币急贬 / 美债收益率急升；
   - 个股前一日放量长上影或主力大幅净流出且连续；
   - 处年内高位(pos>85)且 RSI 超买 → 防高位接盘（呼应用户"高位听消息追买被套"的教训）。
4. **三情景**：给乐观/中性/悲观三套今日区间与应对。
5. **持仓建议落地**：把 `position.advice` 结合预测高低点写成**具体到价位的分批操作清单**。

## 五、与 `stock-tp`（深度研究 skill）协作

早盘 skill 只管"今天/三天怎么操作"。当需要判断"这票中长期基本面/估值/消息面预言到底行不行"时：

- 引用同目录脚本：`python fetch_fundamentals.py <代码>`、`python anspire_search.py ...`；
- 或直接调用老 skill 的 `analyze.py` 得到五因子中长期决策包；
- 把中长期结论作为早盘"趋势偏向"的背景修正（例如基本面极佳则三日趋势判断更偏多）。

二者互不改动、各司其职：`stock-tp` 回答"值不值得拥有"，本 skill 回答"今天怎么打"。
