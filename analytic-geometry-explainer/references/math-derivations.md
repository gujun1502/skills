# 解析推导 · 各切法的闭式来源

> 这套图"智力诚实"的根基：每条显象曲线都来自这里的一个方程。
> 搭新原理时，把"圆+平面"换成你的本体+操作，照同样的步骤解一次。

## 0 · 符号
- 本体：原点处的同心圆族，半径 `r ∈ {r₁,…,rₙ}`（由 `rhythm_radii()` 生成，疏密有律）。
- 圆柱树干：半径 `R`，沿 z 轴高 `H`。年轮 = 圆柱内同心圆沿 z 拉伸成的同心圆筒。
- 切面：竖直面用法向 `(0,1,0)`、偏移 `y=y₀`；斜面用 `z = z₀ + a·x + b·y`。

---

## 1 · 径切（过圆心的竖直面 y = 0）→ 平行直线

圆筒面 `x² + y² = r²` 与平面 `y = 0` 相交：
```
x² = r²  ⟹  x = ±r
```
交线是两条竖直母线（在面板上即一对对称的竖直线）。沿 z 看，**每个年轮筒被侧切成一条平行线**，
相邻线的间距 = 相邻半径差 `Δr`。

> **显象 = 平行线，间距 = 年轮疏密。** 这是最"安静"的纹样：疏密韵律被直接读出。
> 代码里把它铺成水平线（面板坐标），线宽 `lw ∝ Δr` 强化疏密。

---

## 2 · 弦切（偏离圆心的竖直面 y = y₀）→ 嵌套抛物线（山形纹）

圆 `x² + y² = r²` 与平面 `y = y₀` 相交：
```
x² + y₀² = r²  ⟹  x = ±√(r² − y₀²)   （要求 r > |y₀|，否则无交线——如实跳过）
```
半交宽 `x_c(r) = √(r² − y₀²)`。沿树干高度方向把每个年轮筒展开到面板时，
年轮在面板上的"拱起"近似为抛物线：
```
z(x) = z_base(r) + A(r) · ( 1 − (x / x_c)² ),   |x| ≤ x_c
```
- `z_base(r)`：第 r 圈在面板纵向的基准高度（随 r 单调上移）。
- `A(r)`：拱高，随 r 增大略增 → 外圈更舒展。

半径越大 `x_c` 越宽、拱越平；半径接近 `y₀` 时 `x_c→0`，拱成尖顶。
层层嵌套即"山形纹 / 大教堂纹（cathedral grain）"——实木弦切板的真实纹理。

> **为什么是抛物线**：圆被一条不过心的弦切开，弦上各点到圆的"纵向余量"随 x 二次变化
> （勾股 `√(r²−x²)` 在弦邻域的二阶近似）。这是山形纹的解析根源。

---

## 3 · 斜切（与轴成角 θ 的面）→ 拉伸椭圆

把同心圆筒用倾斜角 θ 的平面去截。圆在切面内的投影沿"下坡方向"被拉长：
```
短轴 a = R              （垂直于倾斜方向，不变）
长轴 b = R / cos θ      （沿倾斜方向，被 1/cosθ 拉伸）
```
第 r 圈年轮在切面上是比例椭圆：
```
x(t) = (r/R) · a · cos t
y(t) = (r/R) · b · sin t,   t ∈ [0, 2π)
```
θ→0 时 b→a，退化为正圆（等于横截面，得"标准年轮盘"）；
θ 越大椭圆越扁长。**显象 = 一族同心拉伸椭圆。**

> 这正是圆锥被斜切得椭圆的同一原理（圆锥曲线）——把它落到"年轮被斜切"的语境。
> 与原型 C（参数扫描）相通：扫 θ 即从圆连续过渡到长椭圆。

---

## 4 · 韵律（疏密）的来源 —— 不是随机

年轮疏密用**叠加正弦**调制间距，得到"有机却确定"的节奏：
```
spacing(i) = base + Σ_k  a_k · sin(i · f_k)        # rhythm_radii() 的 harmonics
radii      = cumsum(spacing)
```
默认 `base=0.25`，谐波 `(0.15, 0.6), (0.08, 1.7), (0.05, 3.2)`：
一个慢波（大疏密带）+ 两个快波（细微抖动）。换谐波即换"这棵树的生长史"。

> 关键：用确定的正弦而非 `random`，保证可复现、可解释、可调；
> 真正的随机只作为**极轻噪声**（`0.02·r·sin(3t+r)`）赋予线条"手感"，振幅 < 线宽量级。

## 5 · 推广清单（换原理时照填）

| 你的本体 | 你的操作 | 解什么 | 期望显象 |
|---|---|---|---|
| 同心球 / 经纬 | 平面切 | 球∩平面=圆，纬线∩面=… | 同心圆 / 截线族 |
| 高维体点云 | 投影矩阵 `M_θ` | `P·M_θ` 的边界 | 多方向轮廓影 |
| 含参圆锥曲线 | 扫离心率 e | `r=l/(1+e cosθ)` | 圆→椭→抛→双 |
| 波场 `Σcos(k·x−φ)` | 取不同时刻/截面 | 等相位线 | 干涉条纹族 |
| 图形 + 变换群 | 施加 `T(α)` | `T(α)·P` 不动点 | 对称轨道 |

> 通则：**写出本体方程 → 代入操作 → 解（闭式优先，数值精确次之）→ 得显象采样点 → 画。**
