# 既有建筑改造项目 — 经验与教训总结

> 案例来源：奉化中行支行解危加固+外立面+配套工程（30 年前老楼，整体改造定性）。
> 总结视角：设计院项目负责人 → 业主 / 审图 / 总包 / 施工队四方的真实经验。

---

## 一、立项与定性阶段（最贵的错误发生在这里）

### 教训 1：定性"修缮"还是"整体改造"，是项目所有标准的总开关
- 审图机构会**严格按地方既有建筑改造技术指南**（如《宁波市既有建筑改造消防技术指南》2024.1）判定。
- 一旦被认定"整体改造"，消防、节能、结构、无障碍等强条**必须按现行规范全面补齐**，"历史遗留"四个字救不了你。
- **行动**：报建前主动与审图机构、消防部门预沟通，争取性质书面确认；如能定为"局部修缮"则保留大量原有系统，造价与周期可砍掉一大段。

### 教训 2：可研报告不能简化
- 三份第三方报告是**法定前置文件**，缺一项后续都启动不了：
  1. 消防技术可行性研究报告
  2. 节能诊断报告（启动节能 + 幕墙专项）
  3. 法律效力的地质勘察报告（或合规的引用证明）
- **行动**：立项启动会就把这三家第三方资质单位的招标启动起来，与设计同步，不要等设计院发现缺料再补。

### 教训 3：造价限制 → 项目拆分 → 设备无法闭环
- 业主常因总投资上限把项目拆为"室内装修"+"室外配套"+"外立面"几个标段。
- 给排水、强弱电、消防系统天然跨边界，拆完后**没有任何一个标段图纸可以单独闭环**，审图必然驳回。
- **行动**：立项时即把全楼设备系统作为"打包专项"，单独立项或在最大那个标段里整体设计；外立面 + 节能 + 幕墙必须捆绑。

### 教训 4：副楼范围决策必须在设计前定
- 副楼若不在立项内，其内的核心机房（低压配电室、UPS、消防/生活泵房）必须全部迁主楼。
- 主楼一层结构、面积、流线往往无法承载，**结果就是主楼设计陷入停滞**。
- **行动**：设计合同启动前，让业主书面确认副楼是否纳入；不纳入则同步给出机房替代方案。

## 二、原始资料阶段

### 教训 5：原图只是"勘测参考"，不是"设计依据"
- 30 年前手绘图、无审图章、纸质、可能与现场不符（违章搭建、隐蔽改造）。
- 实例：原图板厚标注 100 mm，**实测仅 60 mm**，地插开槽计划全部推翻；机械凿除停止，全面排查。
- **行动**：合同里就写明"以现场实测为准"，把现状测绘列为强制前置工作；预算包含现状测绘费用。

### 教训 6：隐蔽工程拆开前不要出最终图
- 地下管网、电缆敷设、原基础形式，原图不可信，地面/吊顶未拆都看不到。
- **行动**：分两阶段出图——"先期勘察图"指导拆除，"正式施工图"在拆除完成、隐蔽探查报告齐全后出。

### 教训 7：市政侧资料属于"业主必交项"，设计院不背
- 电力接入点容量/位置、给水接入压力、场地标高、市政管线管廊、消防车道、市政消防水源，**全部需业主向供电、自来水、消防、市政各部门正式调取**。
- **行动**：在第一次设计输入会议上发"业主资料移交清单"，逐项签字确认 deadline，缺一项 → 暂停对应专业出图。

## 三、设计阶段

### 教训 8：消防系统几乎一定要重做
- 90 年代建筑的消防水池容量、水箱、稳压泵、加压送风、排烟、消防电、漏电保护、自动报警、应急照明几乎全不达现行规范。
- **不要试图"局部修补"**，按整体改造一次性补齐反而省钱。
- **行动**：方案阶段就用一张"现行规范 vs 现状"对照表（见 [[01_图纸规范缺项清单]]）和业主对齐，避免后期反复改方案。

### 教训 9：结构加固方案必须现场可施工
- 实例：原设计预应力混凝土管桩，南侧无打桩场地 → 现场施工队建议改为锚杆静压桩，机械占地小、费用低、可实施。
- **行动**：方案阶段邀请总包/施工队提前介入，对桩基、加固方案的可施工性提意见。

### 教训 10：新增大荷载机房必须做相邻结构影响评估
- 新增 252 t 消防水池 + 18 t 高位水箱 + 生活变频泵 + 多联机外机，每一项都是新荷载。
- 影响主楼结构，更可能影响相邻附楼/管网/景观。
- **行动**：设备专业初步选型后立即把荷载点给结构，结构出新增/加固方案后再回设备，**避免最后一步发现景观树/化粪池/管网放不下**。

### 教训 11：节能+幕墙+建筑必须三专业协同审图
- 外立面改造一旦动 → 需出建筑节能报告、幕墙计算书。
- 建筑院流程繁琐，**三专业各审一遍**，不预留时间会卡报审。
- **行动**：项目计划里给建筑+结构+幕墙协同审图留出 2–4 周窗口。

### 教训 12：剖面综合早做不晚做
- 吊顶高度被压死时，桥架、风管、喷淋、新风管抢空间，电气最容易吃亏。
- **行动**：方案/初设阶段就出关键剖面（核心筒、走廊、机房通道），所有专业签字。

## 四、审图与报建阶段

### 教训 13：分阶段报审 vs 审图整体性 — 必有冲突
- 业主想"一阶段先开工结构加固、二阶段再做装修与设备"，但审图机构会要求**一并提交两阶段图纸**才能判断改造规模。
- 不提交 → 暴露前端不合规内容；提交 → 二阶段还没设计完。
- **行动**：报建前先与审图机构预沟通确认是否接受分阶段，如不接受则把二阶段方案级图先做出来"挂靠"，正式版后补。

### 教训 14：审图意见的两种处理路径要分开
- 路径 A：业主有资源能提供资料/做市政整改 → 按现行规范整改。
- 路径 B：业主无法提供 → **必须发函请业主与审图机构正面沟通**，由业主决策"降标 / 申请豁免 / 放弃改造范围"，**设计院不替业主做决定**。
- **行动**：所有审图意见整理成"我方可解决"+"需业主决策"两栏，正式发函业主，避免被动背锅。

### 教训 15：造价增加要在审图前书面预警
- 严格按现行规范整改 → 消防、外墙、防水、节能、幕墙全面追加 → 造价上涨。
- **不要等审图通过后再算账**，业主一定会反弹。
- **行动**：发函业主时同时给"按现行规范执行将带来的造价影响估算"，留痕。

---

## 五、施工配合阶段

### 教训 16：施工队进场后才会暴露真实现场
- 实例 1：污水/雨水接驳井位置与原始测绘不符 → 室外综合管网走向需重调。
- 实例 2：副楼区域燃气管道需改道，**燃气公司协调改道**（不是总包能做的）。
- 实例 3：消防管道由总包改道，必须出方案。
- **行动**：进场首周组织"现场复核会"，所有专业到场，设计-施工-业主三方签字确认偏差。

### 教训 17：拆除是高风险动作，机械参数要验算
- 多机同层 + 机械重量 + 振动 → 楼板穿孔事故。
- **行动**：拆除方案必须含机械参数验算书；首次施工先小面积试拆，确认无异常再扩面。

### 教训 18：景观/树木/化粪池常被最后牺牲
- 设备机房和管网最终落位时，业主"必须保留"的景观树常被牺牲。
- **行动**：方案阶段就把"必须保留项"列入设计输入清单，业主签字；后续若必须动，要业主书面同意。

## 六、可复用的项目动作清单

| 阶段 | 必做动作 | 谁负责 |
|---|---|---|
| 立项 | 性质确认（修缮/改造）书面意见 | 业主+审图预沟通 |
| 立项 | 三份第三方报告招标（消防可研/节能诊断/地勘） | 业主 |
| 立项 | 拆分方案的设备闭环评估 | 设计院总包+业主 |
| 立项 | 副楼范围决策 | 业主 |
| 设计前 | 业主资料移交清单（市政+原档案） | 业主 |
| 设计前 | 现状测绘 + 隐蔽探查计划 | 设计院/第三方 |
| 方案 | 现行规范 vs 现状对照表 | 设计院 |
| 初设 | 关键剖面综合签字 | 全专业 |
| 初设 | 新增荷载对原结构影响评估 | 结构+设备 |
| 报审前 | 分阶段报审策略与审图预沟通 | 设计院+业主 |
| 报审 | 审图意见分类（可解决/需业主决策）+ 造价影响预警 | 设计院 |
| 施工 | 进场首周现场复核会 | 设计+总包+业主 |
| 施工 | 拆除方案 + 机械验算书 | 总包+结构 |
| 施工 | 隐蔽工程报告齐全后出正式图 | 设计院 |

## 七、最容易踩的五个"早知道"

1. **早知道**审图会按地方指南判"整体改造" → 第一次报建前就找审图机构预沟通。
2. **早知道**原图不可信 → 合同里写明现状测绘强制前置。
3. **早知道**造价拆分会让设备闭环不了 → 立项就把设备打包。
4. **早知道**新增机房荷载会牺牲景观 → 设备初步选型就和景观对齐。
5. **早知道**分阶段报审会被卡 → 报建前与审图预沟通策略。

> 一句话：**所有"早知道"的反面，都是"晚知道之后多花的钱"**。
