# 既有建筑改造 — 审批流程要点

> 通用骨架（中国大陆），具体细节看城市差异（见 [[05_城市政策差异]]）。

---

## 一、全流程主线

```
立项 → 可研/三份第三方报告 → 方案设计 → 方案审查
  → 报规（规划审批）→ 施工图设计 → 施工图审查（建审 + 消防 + 节能 + 幕墙）
  → 施工许可 → 施工 → 监督验收 → 消防验收 → 节能验收 → 综合竣工验收 → 备案
```

---

## 二、各环节关键文件

### 2.1 立项
- 项目建议书 / 可研报告
- 项目性质判定（修缮 / 改造 / 整体改造）— **决定后续标准的总开关**
- 原建筑产权与历史档案（消防验收合格证、原审图意见、原图、原结构验收）
- 立项批复（发改委或主管单位）

### 2.2 三份强制第三方报告
- **既有建筑改造消防技术可行性研究报告**（消防大队/审图机构双重要求）
- **节能诊断报告**（启动节能 + 幕墙专项的前置）
- **法律效力的地质勘察报告**（或合规引用证明：参考点 ≤50 m + 地质相似 + 官方证明）
- 其他可能要求：**安全性鉴定报告 + 抗震鉴定报告**（结构改造类必备）

### 2.3 方案审查与报规
- 总平面、立面、效果图（涉及外立面改造的）
- 节能初评、防火初评
- 历史/风貌保护区还需文保局意见

### 2.4 施工图审查
- 建筑、结构、给排水、暖通、电气**全套施工图**
- 计算书（结构、消防水力、加压送风/排烟、负荷、节能、幕墙）
- 三份第三方报告 + 现状测绘 + 加固方案
- 审图机构出具《审查合格证书》

### 2.5 施工许可
- 凭审查合格证书 + 中标通知书 + 监理合同 + 资金到位证明
- 住建局或行政服务中心一站式办理

### 2.6 验收
- 分专业验收 → 综合验收
- 消防、节能单独验收程序（不通过则总验收不通过）
- 涉及幕墙的需幕墙单项验收

## 三、改造类项目特有的流程难点

### 3.1 性质判定的"两次博弈"
- **第一次博弈**（立项阶段）：争取定性"修缮"，可保留大量原系统。
- **第二次博弈**（施工图审查阶段）：审图机构会按地方指南**重新判定**，可能推翻立项时的定性。
- **应对**：立项阶段拿到主管部门书面性质判定，可作为对抗审图重判的依据。

### 3.2 分阶段报审的可行性
- 业主希望"一阶段先开工"以节省工期，但审图机构会要求**整体性判定**。
- 询问当地审图机构是否接受"一阶段报审 + 二阶段挂靠"方案。

### 3.3 副楼/局部不纳入立项的合规性
- 若副楼不在立项范围，相关图纸需从报审文件中删除。
- 但副楼内的核心机房必须迁出（去主楼或新建），需提前规划。

### 3.4 市政条件不足的同步整改
- 若现有市政消防水源/消防车道无法满足改造后需求，需**同步启动市政整改**，并提供证明。
- 这部分常常是工期最大的不确定项。

## 四、各专项审查的核心要点

| 专项 | 主管 | 核心文件 | 易卡点 |
|---|---|---|---|
| 结构 | 审图机构 | 加固计算书、地勘、鉴定报告、桩基方案 | 地勘合规性、新增荷载影响 |
| 建筑 | 审图机构 | 防火分区、疏散、无障碍、节能 | 楼梯前室、自然排烟窗 |
| 消防 | 审图机构 + 消防大队 | 消防水力、报警联动、应急照明、加压送风/排烟 | 水池容量、消防电分离、前端不合规 |
| 节能 | 审图机构 + 节能专项 | 节能诊断报告、热工计算、节能设计 | 外墙保温层缺失、K 值不达标 |
| 幕墙 | 审图机构 + 幕墙专项 | 幕墙计算书（结构、热工、防火、防雷） | 计算书完整性、节能匹配 |
| 给排水 | 审图机构 | 水量水力计算、雨污分流、消防供水 | 消防水池容量、生活/消防分离、雨污混流 |
| 暖通 | 审图机构 | 风量、加压送风、排烟计算 | 防烟楼梯间、机房空间、排烟有效性 |
| 电气 | 审图机构 | 负荷计算、短路计算、火灾报警、应急照明 | 消防电源分离、高压扩容、漏电保护 |
| 防雷 | 气象局/专业机构 | 防雷设计与检测 | 接地电阻、新增金属构件接闪 |
| 人防 | 人防办 | 人防设计或免建证明 | 改造是否触发人防义务 |

## 五、关键时点与典型工期

| 时点 | 典型工期 | 备注 |
|---|---|---|
| 立项到方案审查 | 1–3 月 | 取决于性质判定时间 |
| 三份第三方报告完成 | 1–2 月 | 可与方案并行 |
| 方案 → 施工图 | 2–4 月 | 改造类比新建久 |
| 施工图审查 | 1–2 月 | 平均两次返修 |
| 消防/节能专项审查 | 与施工图同步或后置 | 部分城市并联 |
| 施工 | 6–18 月 | 取决于规模 |
| 综合验收 | 1–3 月 | 含消防、节能单项验收 |

## 六、报审材料"金标准"清单（提交前自检）

- [ ] 立项批复 + 项目性质判定书面文件
- [ ] 业主资料齐全：原产权、原消防验收、原图、市政接入证明、市政管线证明、保护区评估
- [ ] 三份第三方报告：消防可研 / 节能诊断 / 地勘
- [ ] 安全性鉴定报告 + 抗震鉴定报告（主楼、附楼分别）
- [ ] 现状测绘报告（含违章搭建）+ 隐蔽工程探查报告
- [ ] 全专业施工图 + 全专业计算书
- [ ] 加固设计专项 + 新增荷载结构复核
- [ ] 防火分区与疏散计算 + 自然排烟有效面积复核
- [ ] 节能设计 + 幕墙设计（含计算书）
- [ ] 消防系统图（水、电、风全套）
- [ ] 与审图机构的预沟通会议纪要
- [ ] 业主侧资料的书面承诺函（不能提供的部分）

> 缺一项 → 报审会被打回；十项以上缺失 → 直接驳回。
