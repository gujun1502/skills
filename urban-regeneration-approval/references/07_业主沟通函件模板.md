# 业主沟通函件模板（既有建筑改造）

> 用法：模板按"立项→报审→施工"三阶段排列。每份函件应套用设计院抬头、留痕签收。

---

## 模板 1：业主资料移交清单（合同签订后 1 周内发出）

> 目的：把"需要业主协调的事项"集中清单化，避免后期推诿。

```
尊敬的 {业主主体} 项目组：

我方已正式启动 {项目名称} 设计工作。为确保项目顺利推进，依据现行法律法规与行业规范要求，
请贵方按以下清单提供资料，并明确各项资料的预计提交时间。逾期未提供的，相关专业设计工作
将暂停推进，由此产生的工期影响由贵方承担。

资料清单（按业主资料移交清单 A–J 详列）：
  1. 原产权与历史档案
  2. 原消防验收合格证（或等效文件）
  3. 原全套图纸与原审图意见
  ...

请贵方在 {日期} 前书面回复每项资料的预计提交时间。

设计院 {签章}
{日期}
```

## 模板 2：第三方报告资源协调函（立项阶段）

```
尊敬的 {业主主体}：

依据 {本地既有建筑改造消防/抗震/节能技术指南名称}，本项目报审需以下三份第三方报告作为
法定前置文件：
  1. 既有建筑改造消防技术可行性研究报告 — 由具备资质的第三方单位出具
  2. 节能诊断报告 — 由具备资质的第三方检测机构出具
  3. 具备法律效力的地质勘察报告（或合规引用证明）

上述报告由贵方组织资质单位出具，建议在 {日期} 前完成招标与启动，以匹配设计工作节奏。

如贵方无法按期提供，需贵方与审图机构正式沟通并取得书面同意，否则我方报审材料无法形成。

设计院 {签章}
{日期}
```

## 模板 3：副楼范围决策函（设计启动前）

```
尊敬的 {业主主体}：

依据审图机构对 {项目名称} 的初步意见，副楼不在本次立项范围内，相关图纸需从报审文件中
删除。但副楼内现存的核心机房（低压配电室、UPS 电源室、消防泵房与水池、生活泵房与水池）
若不纳入本次设计范围，则必须迁移至主楼 1F 或另寻位置。

请贵方在 {日期} 前书面确认副楼处理方式：
  方案 A：将副楼纳入本次设计与报审范围
  方案 B：副楼不纳入，核心机房迁主楼一层
  方案 C：另寻第三地建设核心机房

不同方案对主楼结构、功能、造价的影响差异显著，请贵方尽快决策，否则主楼设计将陷入停滞。

设计院 {签章}
{日期}
```

## 模板 4：超出设计职责范畴的事项告知函（报审前）

> 对应奉化项目的"中银领导函"。一封最重要的函件。

```
尊敬的 {业主主体} 各位领导：

关于 {项目名称}，依据第一次正式审图意见及我方与审图机构深入沟通的结果，为确保项目顺利
通过审批并保障未来使用的安全性与合规性，现就**超出我方设计院职责范畴、需贵方协调解决**
的关键事项汇报如下：

1. 项目性质界定（核心依据）
   审图机构严格依据 {本地指南} 对本项目进行审核，明确界定本项目为"整体建筑改造"。
   此界定决定后续所有消防/节能/结构标准的适用性。

2. 消防可行性研究报告（前置必备文件）
   必须由具备资质的第三方单位出具；缺乏将导致施工图无法通过消防审查。

3. 节能专项依据
   必须由具备资质的第三方检测机构出具节能诊断报告。

4. 地质勘察报告
   必须为具备法律效力的报告，或提供合规的引用证明（≤50 m 且地质相似）。

5. 消防基础资料
   - 原建筑消防验收合格证明文件
   - 项目周边市政消防设施现状详细资料与容量证明；若不满足改造后需求，需同步启动
     市政消防设施改造。

6. 市政接口与保护区评估
   - 周边市政电力、给水接入点位置/容量/标高
   - 项目场地及周边精确标高
   - 周边所有市政设施情况
   - 若涉及市政划定的重点保护区，需第三方安全评估报告

7. 副楼范围与核心机房选址决策
   {同模板 3}

恳请贵方高度重视以上事项。如确无法提供资料，烦请尽快与审图机构沟通；若严格按现行规范
整改，将对原建筑的消防系统、外墙构造、防水工程、节能设计、幕墙体系等内容进行全面调整，
由此产生的图纸修改及增加将导致工程造价相应增加，预估约 {金额范围}。

设计院 {签章}
{日期}
```

## 模板 5：审图意见分类回函（每轮审图后）

```
尊敬的 {业主主体}：

依据 {日期} 收到的审图意见单，我方将意见分为两类回复如下：

【一】我方可解决意见（共 X 条）— 我方将在 {日期} 前完成修改：
  1. ...
  2. ...

【二】需贵方协调/决策意见（共 X 条）— 请贵方在 {日期} 前书面回复：
  1. {意见编号} 关于市政消防水源容量不足
     需贵方协调市政部门同步整改，或正式申请豁免。
  2. {意见编号} 关于高压配电室扩容
     原前端不具备扩容条件，需贵方决策方案。
  ...

如贵方未在期限内回复，相关意见的审图回复将延迟，由此产生的工期与造价影响由贵方承担。

设计院 {签章}
{日期}
```

## 模板 6：施工现场偏差确认函（进场后）

```
尊敬的 {业主主体}、{总包}：

{项目名称} 施工队进场后，现场复核发现以下与原始测绘 / 设计图纸不符之处，需三方书面确认
处理方案：

1. 污水/雨水接驳井位置与原始测绘不符 → 室外综合管网走向调整方案 {附图}
2. 原结构板厚实测与原图不符 → 拆除工艺调整为小型工具凿除
3. 原燃气管道走向需改道 → 由燃气公司协调，工期影响 {天数}
...

请贵方与总包在 {日期} 前书面回函确认处理方案。未回函视为默认采纳。

设计院 {签章}
{日期}
```

---

## 通用要点

- **抬头**：业主主体写全称（"中国银行股份有限公司 XX 支行"，不要写"中行 XX 行"）。
- **抄送**：监理、总包、咨询、审图机构（如适用）。
- **附件**：所有引用文件以附件形式提供，避免业主"没看到"。
- **签收**：纸质函件需业主签收，电子函件要求 24h 内"已收阅"回复。
- **存档**：邮件 + 纸质 + 内部 OA 三处留存。
