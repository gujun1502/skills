# AI 股票复盘（盘后）

用 akshare 在**收盘后**抓取当日市场全景与个股表现，生成结构化复盘报告，帮助用户总结当天、为次日做准备。

## 适用场景

- "帮我复盘今天的市场" → 大盘 + 板块 + 情绪全景
- "复盘我的持仓：600519、000001、300750" → 多只个股当日表现归因
- "今天涨停/热点板块是哪些？" → 热点与赚钱效应
- "明天该关注什么？" → 基于当日数据给关注清单（非荐股）

## 工作流程

1. **确认范围**：纯大盘复盘 / 大盘 + 用户持仓(让用户给代码列表)。
2. **运行脚本**取数：
   ```bash
   # 大盘全景（主要指数 + 行业板块涨跌 + 资金流 + 涨跌停统计）
   python scripts/review.py --market
   # 个股当日复盘（可多只，逗号分隔）
   python scripts/review.py --holdings 600519,000001,300750
   # 同时来
   python scripts/review.py --market --holdings 600519,000001
   ```
   脚本输出 JSON 到 stdout。
3. **撰写复盘**：由你(Claude)按下面结构组织成中文报告，数据全部来自脚本，不要编造数字。
4. **免责声明**：结尾固定加"以上为公开数据的复盘整理，仅供研究参考，不构成投资建议。"

## 复盘报告结构

1. **一句话定调**：今日市场强/弱、风格(大盘/小盘、成长/价值)。
2. **指数表现**：上证/深成/创业板/科创50/北证50 涨跌幅与成交。
3. **赚钱效应**：涨停家数 / 跌停家数 / 上涨下跌家数比 / 连板高度，判断情绪冷热。
4. **板块轮动**：领涨/领跌行业 Top5，资金净流入/流出 Top 板块 → 当日主线。
5. **持仓归因**(若提供)：每只 涨跌幅、量价、相对所属板块强弱、是否有异动；用一句话说明"跑赢/跑输板块"。
6. **次日关注**：基于今日主线与异动列出 3~5 个**关注方向/观察点**（明确写"关注≠推荐买入"）。

## 情绪判断启发式

| 指标 | 偏强(乐观) | 偏弱(谨慎) |
|---|---|---|
| 涨停家数 | >60 | <30 |
| 跌停家数 | 个位数 | 明显增多 |
| 上涨家数占比 | >60% | <40% |
| 最高连板 | 高位有承接 | 断板、炸板多 |
| 主线资金 | 持续净流入 | 高低切、流出 |

## 注意事项

- akshare 走公网，收盘后数据更新需稍等(通常 15:30 后较稳)；脚本带重试，失败时把报错告诉用户。
- 个股复盘按所属行业对比强弱，行业数据缺失时跳过该项并说明。
- 仅覆盖 A 股(沪深北)。
