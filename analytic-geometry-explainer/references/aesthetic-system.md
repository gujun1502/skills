## 1 · 配色

| 角色 | 色值 | 用处 |
|---|---|---|
| 主奶油底 | `#faf5ee` | 单面纹理图的画布底（`set_facecolor`） |
| 三族极淡底 | `#fff5f0` / `#f0f0ff` / `#f0fff0` | 径切/弦切/斜切板各自的淡背景 |
| 三族主色 | `#8B4513` 暖棕 / `#333399` 靛蓝 / `#2d6b2d` 松绿 | 三种切法的纹样线 |
| 在体高饱和标注 | `red` / `blue` / `green` | 在体定位图里标切面位置（高对比，醒目） |
| 年轮暖金 | `#8B6914` | 木质纹理主线 |
| 顶面深线 | `#5a3e00` | 横截面年轮 |
| 半透外壳 | `#c8a876` @ alpha 0.06–0.12 | 圆柱本体 |
| 轮廓 | `#6b4c1e` / `#8b6c3e` | 顶底圆、母线 |

**配色法则**：一个切法族 = 一个主色，全程不混用。背景用该主色的极淡色调（同色系 5% 饱和），
让纹样"从自己的底里长出来"。整体永远是低饱和暖调，唯一允许高饱和的是"在体定位"里的切面色块。

## 2 · 线条

- **粗细随疏密**：`lw = lw_base + (spacing / max_spacing) · lw_span`（`lw_base=0.4, lw_span≈0.8–1.5`）。
  疏处粗、密处细——让疏密韵律被"重量"二次强化。
- **透明度**：纹样线 `alpha 0.6–0.8`；本体外壳 `0.06–0.15`；标注色块 `0.2–0.3`。
- **生命扰动**：极轻噪声 `0.02·r·sin(3t+r)`，振幅远小于线宽，只为去掉"机器味"。
- 采样密度足够高（每条线 80–200 点）让曲线顺滑，避免折线感。

## 3 · 三维与版式

- **视角**：俯仰 `elev≈12–25`、方位 `azim≈−45…−72`。低俯角看面板纹理最清楚；
  多角度图用 `(15,−75)/(15,−30)/(15,15)/(75,−45)` 四档（左前/正前/右前/俯视）。
- **`ax.axis('off')`**：除"在体定位"保留坐标轴（用于读切面几何），其余一律关轴，
  让纹样独占注意力。
- **半透明叠面**：本体壳、切面色块都用低 alpha 的 `Poly3DCollection` 叠出空间层次。
- **留白**：`bbox_inches='tight'` + 适度 `set_xlim/ylim/zlim` 余量，画面四周要松。
- **标题**：主标题 `fontsize 20–22, fontweight='bold'`；子图标题用该族主色、`pad 10–15`。
  标题写"切法 + 显象"双行（如"弦切面板 / 山形纹·大教堂纹"），点明理与象。

## 4 · 字体与导出

```python
plt.rcParams['font.sans-serif'] = ['Microsoft YaHei', 'SimHei', 'Arial Unicode MS']
plt.rcParams['axes.unicode_minus'] = False
fig.savefig(path, dpi=200, bbox_inches='tight', facecolor='white')
```
- 中文必设，否则方框乱码。
- `dpi=200` 足够印刷级；`facecolor='white'` 保证导出底为白（画布奶油底只在 axes 内）。
- 一张 PNG = 一个认知视图。文件名带序号+语义（`V3_02_三种切面纹理立体.png`）。

## 5 · 规整化图的额外规矩

- 造型板画三面（正/侧/顶）给出厚度，证明它"可造"。
- 必带**尺寸标注**：双箭头 `arrowstyle='<->'` + `{W}m / {H}m`，并抽样标注几处年轮间距值。
- 上排 3D 立体 + 下排 2D 正视，成对出现：一眼看懂"立体长相 + 施工正视"。
