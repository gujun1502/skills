## A. 拆解"预言"四步法（给消息面 0–100 打分）

输入：`anspire_search.fetch()` 返回的 `news.groups`（公司新闻业绩 / 政策监管讲话 /
舆情催化事件 三组，每条含 title/url/content/date/score）。

### 步骤 1：信源核验
- 标注每条的**信源等级**：① 官方(公告/交易所/政府/央行) ② 权威媒体 ③ 券商研报
  ④ 自媒体/论坛(舆情)。等级越低，单独采信权重越小，需交叉印证。
- 标注**时效**：越新越重要；注意旧闻翻炒、标题党。
- 剔除与标的无实质关联、重复、纯营销内容。

### 步骤 2：提取"预言"（消息 → 对股票的隐含断言）
对每条有效消息，写成一句**可证伪的预言**，并标注：
- **方向**：利好 ↑ / 利空 ↓ / 中性 →
- **传导链**：消息 → 影响哪个变量(需求/价格/成本/政策/份额/估值) → 财报哪一项 → 股价。
  例：「出口管制 → 高端芯片断供 → 数据中心收入Q3环比-X% → 杀业绩+杀估值」。
- **弹性(量级)**：对 EPS 或估值的大致影响量级（拍数要标假设，宁可区间）。
- **确定性**：已发生✅ / 进行中⏳ / 传闻❓。
- **兑现时点**：何时能验证（下季报、月度数据、某事件日）。

### 步骤 3：政策 / 领导人讲话专项
- 区分**定调级**(方向信号，影响估值与风险偏好) vs **落地级**(具体补贴/项目/订单，影响业绩)。
- 找**直接受益链**：讲话/政策关键词 → 行业 → 标的主营的命中度（高/中/低/蹭概念）。
- 警惕"利好出尽"：政策已被预期充分定价时，落地反而是兑现卖点。

### 步骤 4：合成消息面分数（0–100）
基线 50，按下列加减后裁剪到 [0,100]，并在报告中列出每条贡献：
- 重大利好且高确定性高弹性：+15~+25；一般利好：+5~+12；利好出尽/已定价：0~+3。
- 重大利空高确定性：-15~-30（黑天鹅见框架"风险否决"）；一般利空：-5~-12。
- 政策定调正向 +5~+12，直接落地受益再 +5~+10；被点名整顿 -15~-25。
- 舆情情绪（自媒体热度）：仅作微调 ±5，不可主导；过热反而提示拥挤。
> 输出：`消息面 = NN 分`，并一句话结论（如"市场关注其 AI 属性与业绩高增长，
> 情绪偏积极，但需消化短期获利盘与主力流出压力"）。
> 把该分数用 `python analyze.py <code> --news-score NN` 回填重算综合分。

## B. 邻近标的比选

挑 1–3 只同业可比公司，各跑一次 `analyze.py`（或至少 market+fundamentals），
列对比表：估值(PE/PB/PS)、营收/净利增速、ROE、毛利率、资金面、消息催化、综合分。
结论回答："同样的预算，为什么是它？"——从 估值性价比 / 增速弹性 / 确定性 /
催化临近度 / 拥挤度 五维给出排序。

## C. 综合分析与预测（第四块）

1. **复述决策表**：把 `decision_(final|preliminary).breakdown` 的
   `因子 × 权重 = 贡献` 原样呈现（这是"可视、可追溯"的核心），并说明若调权重结论是否翻转
   （敏感性：哪个因子是结论的"胜负手"）。
2. **三情景预测**（给概率与价位区间，价位用 `technical_levels`）：
   - 乐观(催化兑现)：触发条件 / 目标位(阻力上沿) / 概率。
   - 中性(震荡)：区间 / 概率。
   - 悲观(风险兑现)：触发条件 / 支撑跌破后下看位 / 概率。
3. **明确结论**：买入 / 卖出 / 持有 + 立场，给**买点、卖点/止盈、止损**（引用点位与 ATR）。
4. **催化剂日历**：未来可能发生、足以改变价值的事件及日期（财报、政策窗口、解禁、
   行业数据、产品发布），标注对应"待验证逻辑"。
5. **操作检查清单**：仓位建议、分批计划、触发加/减仓的客观条件、复盘提醒。
