# AlphaEngine（盘中实时监测）

用 akshare 在**交易时段**抓取实时行情、资金流、板块与涨停数据，计算一组量价/资金启发式信号，帮助用户盯盘与发现异动。

## 适用场景

- "现在 600519 什么情况？" → 个股实时快照 + 分时
- "盘中扫一下异动 / 涨速榜 / 量比榜" → 全市场异动
- "现在哪些板块在拉？资金往哪流？" → 板块资金流榜
- "看看今天的涨停池 / 连板" → 涨停板梳理
- "601318 主力资金净流入多少？" → 个股资金流

## 工作流程

1. **判断是否交易时段**：A 股 09:30–11:30 / 13:00–15:00。非交易时段数据为最近收盘快照，需向用户说明"非盘中，以下为最新收盘/静态数据"。
2. **运行脚本**取数：
   ```bash
   # 个股实时快照 + 当日分时（最近 N 条）
   python scripts/engine.py --symbol 600519
   # 全市场异动榜（按涨速/量比/换手排序，默认涨速）
   python scripts/engine.py --movers --by speed --top 20
   # 板块资金流榜（行业）
   python scripts/engine.py --sectors --top 15
   # 个股资金流（主力/超大单等）
   python scripts/engine.py --fundflow 600519
   # 涨停池
   python scripts/engine.py --zt
   ```
   脚本输出 JSON 到 stdout，已附带计算好的信号字段。
3. **信号解读**：由你(Claude)结合下表把数据翻译成人话，标注"强/中/弱"与可能含义，但**不下买卖结论**。
4. **汇总输出**：个股给"快照+信号+一句话状态"；榜单给 Top 列表 + 2~3 条"值得注意的异动"。
5. **免责声明**：结尾固定加"以上为实时数据的启发式信号，仅供研究参考，不构成投资建议；盘中数据有延迟与波动。"

## 信号启发式

| 信号 | 含义 | 强信号参考 |
|---|---|---|
| 量比 (volume_ratio) | 当前成交量 vs 近期均量 | >2 放量，>5 显著异动 |
| 涨速 (speed/涨跌幅) | 短时价格变动 | 分钟级涨速高=资金快速进出 |
| 换手率 (turnover) | 活跃度 | 中小盘 >10% 偏高 |
| 主力净流入 (main_net) | 大单+超大单净额 | 持续为正且占比高=资金认可 |
| 板块净流入 | 行业资金方向 | 榜首板块=当下主线候选 |
| 连板数 (limit_count) | 涨停接力强度 | 高位连板=情绪标的，风险高 |

> 信号互相印证才更可靠：价涨 + 放量 + 主力净流入同向，比单一指标更有意义。高位连板/异动往往伴随高风险，务必提示用户。

## 注意事项

- akshare 实时接口走公网，盘中高峰可能延迟数十秒或偶发失败；脚本带重试，失败时把报错告诉用户。
- 实时快照接口数据量大(全市场数千只)，脚本默认只返回排序后的 Top N，避免刷屏。
- 仅覆盖 A 股(沪深北)。
